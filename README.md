# Chess Website

An interactive web-based chess application where users can play chess in their browser.  
The project implements the core rules of chess and includes an experimental bot opponent.

## Features

- Interactive chessboard
- Legal move validation
- Turn-based gameplay
- Piece movement animations
- Game reset functionality
- Basic chess bot

## Bot

A basic chess bot has been implemented that selects **random legal moves** from the available move set.  
This serves as the foundation for a more advanced AI that may use evaluation functions or search algorithms in the future.
